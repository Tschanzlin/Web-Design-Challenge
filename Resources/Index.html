Welcome to Weather Py Results

This analysis looks at weather patterns across over 500 cities, selected randomly, and located globally. Locational data for each city was generated using a random number generator that created a list of lat / long locations. Each location was then used to select the nearest city to that lat/long data point from the citypy Python library.

Once the city information was prepared, weather information from each city was obtained using the Open Weather API. For each city, the tempurature, humidity, cloudiness, and windspeed were gathered for a single day as of the data of the query (July 2020). This information was then plotted to determine whether there was any correlation between city latitutde and weather condition.

As the enclosed graphs will show, there is a strong correlation between latitude and tempurature, but little correlation with the other measured weather conditions.
